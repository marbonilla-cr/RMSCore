B) computeDailyPayrollV2 reemplaza lógica antigua
Inputs:
{ punchesForDay, scheduleForDay, dailyRate, hrConfig, dateStr }
(Phase A NO incluye aprobación persistente todavía. Si overtimeRequiresApproval=true, overtimePaid=0)

Caso 1: No schedule (o dayOff) pero punch existe:
- paidStart = firstClockIn
- ordinaryEnd = lastClockOut (si falta, aplicar autocierre determinista: firstClockIn + overtimeDailyThresholdHours horas)
- normalMinutesRaw = max(0, ordinaryEnd - paidStart)
- overtimeCalculatedMinutes = 0
- flags += ["SIN_HORARIO"]

Caso 2: Schedule existe:
- paidStart:
  if paidStartPolicy == 'SCHEDULE_START_CAP':
    paidStart = max(firstClockIn, scheduledStart)
  else:
    paidStart = firstClockIn
- ordinaryEnd = min(lastClockOut, scheduledEnd)
- normalMinutesRaw = max(0, ordinaryEnd - paidStart)

Overtime:
- overtimeCalculatedMinutes = 0
- SOLO si lastClockOutType == MANUAL y lastClockOut > scheduledEnd:
    overtimeCalculatedMinutes = lastClockOut - scheduledEnd
- Si lastClockOutType in [AUTO, AUTO_NO_SCHEDULE, GEO_AUTO] => overtimeCalculatedMinutes = 0 siempre

Overtime pagado (Phase A):
- if overtimeRequiresApproval AND overtimeCalculatedMinutes > 0:
    overtimePaidMinutes = 0
    flags += ["OVERTIME_PENDIENTE_APROBACION"]
  else:
    overtimePaidMinutes = overtimeCalculatedMinutes

Break deduction:
- if breakDeductEnabled:
    payableWindow = normalMinutesRaw + overtimeCalculatedMinutes
    if payableWindow >= breakThresholdMinutes:
        unpaidBreakMinutes = breakDeductMinutes
    else 0
  else unpaidBreakMinutes=0
- Aplicar solo a ordinario:
  normalMinutesPaid = max(0, normalMinutesRaw - unpaidBreakMinutes)
  overtimePaidMinutes se mantiene igual (no se descuenta)

Tardiness:
- if scheduledStart exists AND firstClockIn > scheduledStart + latenessGraceMinutes:
    tardyMinutes = firstClockIn - scheduledStart
    flags += ["TARDE"]
- (descuento de pago por tardanza ya queda reflejado por paidStart=max(clockIn,schedStart) cuando llega tarde)

Pay:
- hourlyRate = dailyRate / overtimeDailyThresholdHours
- normalPay = round2((normalMinutesPaid/60) * hourlyRate)
- overtimePay = round2((overtimePaidMinutes/60) * hourlyRate * overtimeMultiplier)
- basePay = normalPay + overtimePay
- paidWorkedMinutes = normalMinutesPaid + overtimePaidMinutes

Return DailyPayrollResult con:
normalMinutesRaw, normalMinutesPaid,
overtimeCalculatedMinutes, overtimePaidMinutes,
unpaidBreakMinutes, paidWorkedMinutes, tardyMinutes,
normalPay, overtimePay, basePay, hourlyRate,
scheduledStartTime, scheduledEndTime,
flags: string[],
booleans legacy: autoCheckout, noShow, late, hasMultiplePunches

Notas obligatorias:
- Retroactivo determinista: si punch abierto en fecha pasada, NO usar now. Autocerrar virtualmente a scheduledEnd o clockIn+autoCloseHours según corresponda y marcar flag AUTO_OUT/AUTO_NO_SCHEDULE.
- lastClockOutType debe tomarse del último punch real; si no existe, inferir AUTO.
