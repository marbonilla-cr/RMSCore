4.3 computeDailyPayrollV2 (reemplaza la lógica antigua)
Inputs:
{ punchesForDay, scheduleForDay, employee(dailyRate, role, locationId...), hrSettings, overtimeApprovedForRange(boolean) }
Outputs:
{ normalMinutesRaw, overtimeCalculatedMinutes, unpaidBreakMinutes, normalMinutesPaid, overtimeMinutesPaid,
  paidWorkedMinutes, tardyMinutes, normalPay, overtimePay, basePay,
  flags: [ 'TARDE', 'AUTO_OUT', 'GEO_AUTO', 'SIN_HORARIO', 'PUNCH_BASURA_FILTRADO', ... ],
  clockInType/clockOutType resumidos }

Reglas:
A) Determinar scheduledStartAt / scheduledEndAt (si existe y no dayOff)
B) Determinar clockIn/clockOut efectivos:
- Usar primer clockIn y último clockOut de punches normalizados del día
- Si falta clockOut:
  - si hay scheduledEndAt: workedEnd=scheduledEndAt, flag AUTO_OUT
  - si no hay schedule: workedEnd = min(now, clockIn+autoCloseHours) SOLO para vista; idealmente el job lo cerrará, pero el reporte debe ser estable (no usar now si es reporte histórico; si fecha pasada y sigue abierto, cerrar virtualmente a clockIn+autoCloseHours).
  - clockOutType virtual = AUTO_NO_SCHEDULE

C) paidStartPolicy:
- if SCHEDULE_START_CAP and scheduleStart exists:
    paidStart = max(clockIn, scheduledStartAt)
- else:
    paidStart = clockIn
Esto hace que tardanza descuente pago.

D) Fin ordinario:
- if scheduleEnd exists:
    ordinaryEnd = min(clockOut, scheduledEndAt)
  else:
    ordinaryEnd = clockOut
- normalMinutesRaw = max(0, ordinaryEnd - paidStart)

E) Overtime calculado (solo por horario y solo salida MANUAL real)
- overtimeCalculatedMinutes = 0
- if scheduleEnd exists and clockOutType in ['MANUAL','MANUAL_OUTSIDE'] and clockOut > scheduledEndAt:
    overtimeCalculatedMinutes = clockOut - scheduledEndAt
- if clockOutType in ['AUTO','AUTO_NO_SCHEDULE','GEO_AUTO']: overtimeCalculatedMinutes = 0

F) Break deduction (>9h) (si breakDeductEnabled)
- workedTotalRaw = normalMinutesRaw + overtimeCalculatedMinutes
- unpaidBreakMinutes = (workedTotalRaw >= breakThresholdMinutes) ? breakDeductMinutes : 0
- if breakApplyTo='NORMAL_ONLY':
    normalMinutesPaid = max(0, normalMinutesRaw - unpaidBreakMinutes)
    overtimePaidCandidate = overtimeCalculatedMinutes
  else:
    (no implementar otros modos en v1)
- Overtime pagado depende de aprobación:
  - if overtimeRequiresApproval:
      overtimeMinutesPaid = overtimeApprovedForRange ? overtimePaidCandidate : 0
    else:
      overtimeMinutesPaid = overtimePaidCandidate

G) Tardanza (indicador)
- if scheduleStart exists and clockIn > scheduledStartAt + latenessGraceMinutes:
    tardyMinutes = clockIn - scheduledStartAt
  else 0

H) Pago
- hourlyRate = dailyRate / hrSettings.overtimeDailyThresholdHours
- normalPay = round2((normalMinutesPaid/60)*hourlyRate)
- overtimePay = round2((overtimeMinutesPaid/60)*hourlyRate*hrSettings.overtimeMultiplier)
- basePay = normalPay + overtimePay
- paidWorkedMinutes = normalMinutesPaid + overtimeMinutesPaid

4.5 CCSS / Cargas sociales (retroactivo)
