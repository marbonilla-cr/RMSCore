T003: Deducción automática de almuerzo/café SOLO si trabajó 9+ horas (sin tocar overtime)
FILES: server/payroll.ts, client/src/pages/hr/reports.tsx

REGLA NEGOCIO
- Si workedMinutes >= 540 (9h): unpaidBreakMinutes = 60
- Si workedMinutes < 540: unpaidBreakMinutes = 0
- NO aplicar entre 8 y 9; NO aplicar si <8.
- Mostrar descuento en planilla UI.

IMPLEMENTACIÓN EN computeDailyPayroll (o motor equivalente)
PASOS (orden obligatorio):
1) Calcular workedMinutes (raw) desde clockIn hasta workedEnd (con autoCheckout si falta salida).
2) Calcular overtimeMinutes basado en horario (si hay scheduleEnd y clockOutType=MANUAL):
   overtimeMinutes = max(0, clockOut - scheduledEndAt) else 0
3) normalMinutesRaw = max(0, workedMinutes - overtimeMinutes)
4) unpaidBreakMinutes = (workedMinutes >= 540) ? 60 : 0
5) Aplicar el descuento SOLO a lo ordinario:
   normalMinutesPaid = max(0, normalMinutesRaw - unpaidBreakMinutes)
   overtimeMinutesPaid = overtimeMinutes (NO se reduce)
6) Calcular pagos usando normalMinutesPaid y overtimeMinutesPaid:
   - hourlyRate = dailyRate / Number(hrConfig.overtimeDailyThresholdHours || 8)
   - normalPay = round2((normalMinutesPaid/60) * hourlyRate)
   - overtimePay = round2((overtimeMinutesPaid/60) * hourlyRate * Number(hrConfig.overtimeMultiplier || 1.5))
7) Exponer unpaidBreakMinutes en el resultado del día para UI.

UI
- Agregar columna "Desc." (minutos) en la planilla (y/o daily breakdown) para mostrar unpaidBreakMinutes.
- Asegurar que totales y desglose sumen correctamente.

ACCEPTANCE
- Si workedMinutes >= 540 => descuenta 60 min del ordinario.
- Si workedMinutes < 540 => descuenta 0.
- overtime nunca se reduce por la deducción.
- UI muestra el descuento.
