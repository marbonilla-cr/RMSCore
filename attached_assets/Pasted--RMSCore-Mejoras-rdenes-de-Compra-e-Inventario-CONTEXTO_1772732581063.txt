# RMSCore — Mejoras Órdenes de Compra e Inventario

## CONTEXTO CRÍTICO — LEER ANTES DE MODIFICAR

- Todas las rutas de inventario usan el prefijo `/api/inv/` (NO `/api/inventory/`)
- Rutas en: `server/inventory-routes.ts` dentro de `registerInventoryRoutes()`
- Storage en: `server/inventory-storage.ts` (funciones exportadas, no clase)
- Tabla de usuarios: `users` con campo `display_name` (schema: `users.displayName`)
- Los campos numéricos de líneas de OC vienen como `string` (tipo `numeric` de Drizzle):
  - Cantidad: `qtyPurchaseUom`
  - Precio unitario: `unitPricePerPurchaseUom`
  - Cantidad recibida (en base UOM): `qtyBaseReceived`
- Nombre del artículo en líneas ya viene incluido como `invItemName` (join existente)
- `invPoReceipts` usa `receivedByEmployeeId` (no `receivedByUserId`)
- `invPoReceiptLines` usa `receiptId` (no `poReceiptId`) y `poLineId` (no `purchaseOrderLineId`)
- El campo de nota en `invPoReceipts` es `note` (no `notes`)
- Función de storage para líneas de recepción: `getPoReceiptLines(receiptId)` — ya hace
  join con `invPurchaseOrderLines` e `invItems`, devuelve `invItemName` directamente
- Función de storage para recepciones: `getPoReceipts(purchaseOrderId)` — existe pero
  solo devuelve datos crudos de `inv_po_receipts`, SIN líneas ni nombre de empleado

## T002 — Descarga CSV de OC

**Archivo:** `client/src/pages/inventory/purchase-orders.tsx`

Agregar esta función dentro del componente, junto a los demás handlers:

```typescript
function handleDownloadCSV() {
  if (!selectedPO) return;
  const BOM = "\uFEFF";
  const headers = [
    "Artículo", "Cantidad", "UOM compra", "Precio Unitario",
    "Subtotal", "Recibido (base UOM)", "Estado línea"
  ].join(",");

  const rows = (selectedPO.lines ?? []).map(line => {
    const qty = parseFloat(line.qtyPurchaseUom);
    const price = parseFloat(line.unitPricePerPurchaseUom);
    const subtotal = qty * price;
    return [
      line.invItemName ?? "",
      qty.toLocaleString("es-CR"),
      line.purchaseUom,
      price.toLocaleString("es-CR", { minimumFractionDigits: 2 }),
      subtotal.toLocaleString("es-CR", { minimumFractionDigits: 2 }),
      parseFloat(line.qtyBaseReceived).toLocaleString("es-CR"),
      line.lineStatus,
    ].map(v => `"${String(v).replace(/"/g, '""')}"`).join(",");
  });

  const total = (selectedPO.lines ?? []).reduce(
    (s, l) => s + parseFloat(l.qtyPurchaseUom) * parseFloat(l.unitPricePerPurchaseUom), 0
  );
  const totalRow = [
    `"Total"`, `""`, `""`, `""`,
    `"${total.toLocaleString("es-CR", { minimumFractionDigits: 2 })}"`,
    `""`, `""`
  ].join(",");

  const csv = BOM + [headers, ...rows, totalRow].join("\n");
  const blob = new Blob([csv], { type: "text/csv;charset=utf-8;" });
  const url = URL.createObjectURL(blob);
  const a = document.createElement("a");
  a.href = url;
  a.download = `OC-${selectedPO.id}.csv`;
  document.body.appendChild(a);
  a.click();
  document.body.removeChild(a);
  URL.revokeObjectURL(url);
}
```

Agregar botón en el área de acciones del detalle de la OC:
```tsx
<Button variant="outline" size="sm" onClick={handleDownloadCSV}>
  Descargar CSV
</Button>
```

**Aceptación:** El botón genera `OC-{id}.csv` con encabezados en español,
todas las líneas y fila de total. Abre en Excel sin problemas de encoding.

El endpoint `GET /api/inv/purchase-orders/:id/receipts` ya existe en

```typescript
app.get("/api/inv/purchase-orders/:id/receipts", requirePermission("INV_MANAGE_PO"), async (req, res) => {
  try {
    res.json(await invStorage.getPoReceipts(parseInt(req.params.id)));

  }

// DESPUÉS:
app.get("/api/inv/purchase-orders/:id/receipts", requirePermission("INV_MANAGE_PO"), async (req, res) => {
  try {
    const poId = parseInt(req.params.id);
    const receipts = await invStorage.getPoReceipts(poId);

    // Para cada recepción, cargar sus líneas y el nombre del empleado
    const result = await Promise.all(receipts.map(async (receipt) => {
      const lines = await invStorage.getPoReceiptLines(receipt.id);

      // Buscar el displayName del empleado en la tabla users
      let receivedByName = "—";
      try {
        const { db } = await import("./db");
        const { users } = await import("@shared/schema");
        const { eq } = await import("drizzle-orm");
        const [user] = await db
          .select({ displayName: users.displayName })
          .from(users)
          .where(eq(users.id, receipt.receivedByEmployeeId))
          .limit(1);
        if (user) receivedByName = user.displayName;
      } catch (_) {}

      return {
        id: receipt.id,
        receivedAt: receipt.receivedAt,
        receivedByName,
        note: receipt.note ?? null,
        lines,   // cada línea ya tiene invItemName, qtyPurchaseUomReceived, unitPricePerPurchaseUom, purchaseUom
      };
    }));

    res.json(result);
  } catch (err: any) {
    res.status(500).json({ message: err.message });
  }
});
```

### T005-B: Función centralizada de guardado

```typescript
const saveChanges = useCallback(async () => {
  if (!isDirty || Object.keys(pendingRef.current).length === 0) return;
  setIsSaving(true);
  setSaveStatus("saving");
  const payload = { ...pendingRef.current };
  try {
    await apiRequest("PATCH", `/api/inv/items/${item.id}`, payload);
    queryClient.invalidateQueries({ queryKey: ["/api/inv/items"] });
    pendingRef.current = {};
    setIsDirty(false);
    setSaveStatus("saved");
    // Volver a idle después de 2 segundos
    setTimeout(() => setSaveStatus("idle"), 2000);
  } catch {
    setSaveStatus("error");
    toast({
      variant: "destructive",
      title: "Error al guardar",
      description: "No se pudieron guardar los cambios. Intentá de nuevo.",
    });
  } finally {
    setIsSaving(false);
  }
}, [isDirty, item.id]);
```
