## Prompt 2 de 2 — Venta Rápida

**Contexto:** Agregar un botón "Venta Rápida" en la pantalla de mesas, centrado en la barra de búsqueda. Abre un flujo igual al de una mesa normal pero sin mesa física — pide un nombre libre, permite agregar items, los items marcados para KDS van al KDS, y se cobra inmediatamente con el PayDialog existente.

---

### PARTE A — Base de datos

Agregar en `migrations/0021_quick_sales.sql`:

```sql
--> statement-breakpoint
ALTER TABLE "orders" ADD COLUMN IF NOT EXISTS "is_quick_sale" boolean NOT NULL DEFAULT false;
--> statement-breakpoint
ALTER TABLE "orders" ADD COLUMN IF NOT EXISTS "quick_sale_name" varchar(100);
```

Agregar en `shared/schema.ts` en la tabla `orders`:
```typescript
isQuickSale: boolean("is_quick_sale").notNull().default(false),
quickSaleName: varchar("quick_sale_name", { length: 100 }),
```

---

### PARTE B — Backend en `server/routes.ts`

**1. Endpoint crear venta rápida:**
```typescript
app.post("/api/waiter/quick-sale", requireAuth, async (req, res) => {
  try {
    const { name } = req.body;
    if (!name?.trim()) return res.status(400).json({ message: "Nombre requerido" });

    const businessDate = await getBusinessDate(req.tenantSchema);
    const dailyNumber = await getNextDailyOrderNumber(req.db, businessDate);
    const globalNumber = await getNextGlobalOrderNumber(req.db);

    const [order] = await req.db.insert(orders).values({
      tableId: null,
      status: "OPEN",
      isQuickSale: true,
      quickSaleName: name.trim(),
      businessDate,
      dailyNumber,
      globalNumber,
      openedAt: new Date(),
      responsibleWaiterId: req.session?.userId ?? null,
    }).returning();

    res.status(201).json(order);
  } catch (err: any) {
    res.status(500).json({ message: err.message });
  }
});
```

**2. Verificar que los endpoints existentes de mesa funcionan con `tableId = null`:**
- `POST /api/waiter/tables/:id/send-round` — este usa `tableId`, para ventas rápidas usar el `orderId` directamente
- Agregar endpoint alternativo:
```typescript
app.post("/api/waiter/orders/:orderId/send-round", requireAuth, async (req, res) => {
  // Misma lógica que send-round pero recibe orderId directamente
  // sin buscar mesa — para ventas rápidas
});
```

**3. En `GET /api/pos/tables`** — incluir ventas rápidas abiertas junto con las mesas normales, con `tableName = order.quickSaleName` y un indicador `isQuickSale: true`.
