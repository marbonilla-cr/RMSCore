**CAMBIO 3 — QR de despacho en Admin → Configuración**

En `client/src/pages/admin/business-config.tsx`, en la sección de modos de operación, cuando `operationModeDispatch` está activo mostrar:

```
[toggle] Despacho en Mostrador ✅

URL de despacho:
https://rms.restlaantigua.com/qr/DISPATCH

[Descargar QR] ← botón que genera y descarga el QR como PNG
```

La URL del QR de despacho es la misma URL del QR de mesa pero con un código especial `DISPATCH` que indica que no hay mesa física — el sistema lo trata como orden de despacho automáticamente.

Para generar el QR en el frontend usar la librería `qrcode` — instalar si no existe:
```bash
npm install qrcode @types/qrcode
```

El botón genera un canvas con el QR y lo descarga como `qr-despacho-${businessName}.png`.
