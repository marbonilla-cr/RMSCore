REGLAS BASE DE CÁLCULO (POR DÍA)
Inputs del día: schedule { scheduledStartAt, scheduledEndAt }, punches del día (pueden ser múltiples), user { dailyRate }.
- Derivar hourlyRate = dailyRate / 8

Política múltiples punches mismo día:
- Tomar primer clock_in y último clock_out del día.
- Marcas intermedias: registrar audit/log (no se pierden; al menos quedan auditadas).

Tardía (amonestación):
- late = clock_in > scheduledStartAt
- tardyMinutes = max(0, clock_in - scheduledStartAt)

No show (amonestación):
- Si existe schedule para ese día y NO existe clock_in => noShow = true
- Si noShow = true, pago base del día = 0.

Worked time:
- workedStart = clock_in
- workedEnd:
  - Si clock_out es MANUAL/REAL: workedEnd = clock_out
  - Si falta clock_out (AUTO / cierre por sistema): workedEnd = scheduledEndAt (obligatorio)
- workedMinutes = max(0, workedEnd - workedStart)

Overtime (CRÍTICO: NO usar umbral fijo):
- Si clock_out es MANUAL/REAL y scheduledEndAt existe:
    overtimeMinutes = max(0, clock_out - scheduledEndAt)
  Si clock_out es AUTO o clock_out <= scheduledEndAt:
    overtimeMinutes = 0
- normalMinutes = workedMinutes - overtimeMinutes
- normalPay = (normalMinutes/60) * hourlyRate
- overtimePay = (overtimeMinutes/60) * hourlyRate * 1.5
- baseTotalPayDay = normalPay + overtimePay

Early arrival:
- Si clock_in < scheduledStartAt: tardía 0, se paga desde clock_in igualmente.

Turnos cruzando medianoche:
- Debe soportarse (ej: 18:00–02:00). Comparar con datetime real en TZ local.

PRORRATEO DEL SERVICIO POR TIEMPO (por día)
Para cada día D dentro del rango:
1) Elegibles(D) = saloneros elegibles que trabajaron ese día (workedMinutes > 0).
2) weight(i,D) = workedMinutes(i,D) / sum(workedMinutes(all eligibles, D))
3) Percent por empleado:
   percent(i) = employeeOverridePercent ?? hrServicePercentDefault  (solo si override existe en schema)
4) Si percent varía por empleado (override existe):
   - NO uses un único pool con un solo percent.
   - Calcula por empleado:
     serviceRaw(i,D) = ventas_base(D) * percent(i)  (o proporcional sobre service_amount si existe y quieres mantener override: documenta)
     servicePay(i,D) = serviceRaw(i,D) * weight(i,D)
5) Si percent NO varía (solo default):
   - servicePay(i,D) = poolService(D) * weight(i,D)

Nota: si no hay elegibles(D), servicePay=0 y documentar.
