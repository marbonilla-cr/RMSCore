   - borrar: if (emp.role !== "WAITER" && emp.role !== "SALONERO") continue

2) Construir eligibleIdsByDate desde serviceLedgerByEmployee:
   - serviceLedgerByEmployee: Record<dateStr, Record<employeeId, amount>>
   - eligibleIdsByDate[dateStr] = Set<number>(
       Object.keys(serviceLedgerByEmployee[dateStr]).map(Number).filter(id => id > 0)
     )
   - NO incluir 0 (null/no asignado)

3) Para cada dateStr del rango:
   - eligibleIds = eligibleIdsByDate[dateStr] (puede ser empty)
   - unassignedDay = serviceLedgerByEmployee[dateStr]?.[0] ?? 0 (si routes lo usa), o parámetro unassigned separado
   - poolDay = sum(amounts de empId>0) + unassignedDay

   Reglas de asignación:
   a) Si poolDay <= 0 => allocationModeByDate[dateStr] = "UNASSIGNED" (no hay servicio ese día)
   b) Si eligibleIds vacío y poolDay > 0:
      - todo queda como no asignado (no repartir)
      - allocationModeByDate[dateStr] = "UNASSIGNED"
      - acumular serviceUnassignedTotal += poolDay
   c) Si hay punches/minutos:
      - calcular paidWorkedMinutes por empId (ya existe en engine)
      - eligiblesConMin = eligibleIds con paidWorkedMinutes>0
      - si sumMin > 0:
          repartir SOLO entre eligiblesConMin:
            weight = paidMin(emp)/sumMin
            pay = round2(poolDay * weight)
          allocationModeByDate[dateStr] = "HOURS_PRORATED"
        else:
          fallback ledger direct (d)
   d) Fallback LEDGER_DIRECT (si no hay punches o sumMin==0):
      - para cada empId en eligibleIds:
          result[empId] += serviceLedgerByEmployee[dateStr][empId]
      - allocationModeByDate[dateStr] = "LEDGER_DIRECT"
      - además, acumular serviceUnassignedTotal += unassignedDay (si existe)

4) Retornar allocationModeByDate: Record<string, "HOURS_PRORATED"|"LEDGER_DIRECT"|"UNASSIGNED">
   y serviceUnassignedTotal (ya existe o agregarlo si falta).
